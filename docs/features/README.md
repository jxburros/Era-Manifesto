## ðŸŽ¯ Core Features

Era Manifesto includes:

1. **Today Dashboard** - Daily task and event overview
2. **Timeline View** - Chronological project timeline
3. **Calendar View** - Calendar-based planning
4. **Task Management** - Create, organize, and track tasks
5. **Event Tracking** - Schedule and manage events
6. **Financial Tracking** - Budget and expense management
7. **Progress Monitoring** - Visual progress indicators
8. **Mobile/PWA Support** - Install as mobile app
9. **Offline-First** - Works without internet
10. **Firebase Sync** - Cloud synchronization
